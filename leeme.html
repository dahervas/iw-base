Comentarios al profesor

Para que en la base de datos haya productos, colecciones y alguna imagen asociada, es necesario pasar por la vista /bd. Si no te apetece arrancar la visita de la página de la /bd ahora salen unas pantallas bastante amigables, en la vista /home y /collections si no hay nada en la base de datos. Te preguntará si quieres introducir datos. Al seleccionar nuevo producto en /home te creará el producto, almacenará en la carpeta tmp/iw/product la foto pero no la mostrará (falta arreglarlo). Aún no se pueden subir colecciones.

Vistas

  • Base de datos --> Carga datos en la base de datos para que todo funcione correctamente.
  • Página principal --> Home, vista donde salen todos los productos insertados en la base de datos. Si no hay nada en la BD saldrá una pantalla amigable.
  • Colecciones --> Muestra las coleciones insertadas en la base de datos. Si no hay nada en la BD saldrá una pantalla amigable.
  • Producto --> En esta página se podra observar la información sobre un producto determinado, es decir, imagen de producto, nombre, detalles, cantidad del producto a disposición del usuario... --> Se accede desde el home pulsando a cualquier producto.
  • Búsqueda --> La página de búsqueda tiene una interfaz muy bonita. Ahora sólo hace falta que cumpla su cometido, que realice búsquedas en la base de datos.
  • Nueva colección --> Página para añadir una nueva colección (aún sin funcionar)
  • Nuevo producto --> Página para añadir un producto. Contiene un formulario para subir una foto y la información del producto. De momento almacena la foto en la carpeta selecionada pero no la muestra cuando buscas el producto.
  • Perfil --> Página del perfil del usuario (no recibe aún el dato de la sesión)
