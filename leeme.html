Comentarios al profesor

Para que en la base de datos haya productos, colecciones y alguna imagen asociada, es necesario pasar por la vista /bd

Vistas

  • Base de datos --> Carga datos en la base de datos para que todo funcione correctamente.
  • Página principal --> Home, vista donde salen todos los productos insertados en la base de datos
  • Colecciones --> Muestra las coleciones insertadas en la base de datos
  • Producto --> Se accede desde el home pulsando a cualquier producto.
  • Búsqueda --> Página de búsqueda.
  • Nueva colección --> Página para añadir una nueva colección (aún sin funcionar)
  • Nuevo producto --> Página para añadir un producto (aún sin funcionar)
  • Perfil --> Página del perfil del usuario (no recibe aún el dato de la sesión)
