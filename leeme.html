Tamaa

Tamaa es un proyecto con el fin de hacer crecer el altruismo en el mundo. Es una página para compartir todas nuestras cosas.

Comentarios al profesor

  • El import.sql introducirá varios usuarios en la base de datos pero vacíos. Para hacer las pruebas habrá que crear nuevos productos, así como colecciones
  • Para añadir un producto existente a una colección existente bastará con pinchar en el icono home, luego en la "X" al lado del nombre de nuestro producto y elegir la colección a la que deseamos añadir el producto.

    Vistas

    Usuario 'USER'
    • Página principal Home, vista donde salen todos los productos insertados en la base de datos. Si no hay nada en la BD saldrá una pantalla amigable. Pulsando a los productos veremos información sobre ellos. También aparecerá un + a la derecha del nombre del producto, esto nos permitirá añadir el producto a una colección existente.
    • Colecciones Muestra las coleciones insertadas en la base de datos. Si no hay nada en la BD saldrá una pantalla amigable. Al pulsar sobre una colección nos mostrará más información sobre ella.
    • Producto En esta página se podrá ver la información sobre un producto determinado, es decir, imagen de producto, nombre, detalles, cantidad del producto a disposición del usuario, si está prestado en ese momento o no. También se puede eliminar ese producto o votarlo pulsando en las estrellas de valoración y se puede comentar el estado o lo que nos apetezca (sin que al moderador le parezca mal nuestro comentario). Desde esta pantalla también podremos agregar el producto a la colección y podremos pedirlo prestado si no está ocupado.
    • Búsqueda En la página de búsqueda, se introduce el nombre de un producto que se quiera buscar y mostrará su imagen principal (dependiendo del tamaño) con su nombre con link para acceder a la página de dicho producto.
    • Nueva colección Página para añadir una nueva colección. Se puede acceder a ella desde la página de colecciones si no existe ninguna en el sistema o podemos acceder a través de un botón en la página de perfil.
    • Nuevo producto Página para añadir un producto. Contiene un formulario para subir una foto principal, varias fotos más y la información del producto. Estas imágenes son almacenadas en /tmp/iw. Al igual que con las nuevas colecciones se podrá acceder al formulario para añadir un producto desde la página donde se muestran todos los productos home, o desde el perfil de cada usuario.
    • Perfil Página del perfil del usuario donde aparecen todos sus productos y colleciones. En esta página el usuario puede añadir productos y crear colleciones. También puede modificar su perfil.
    • Perfil de otro usuario Esta es una página destinada a mostar la información de otro usuario. Sus productos y sus colecciones, también se podrá valorar al usuario con las estrellas, así cualquiera sabe si está haciendo bien en prestarle un objeto a alguien. Si intentas entrar en tu propio perfil a través de esta dirección, la página te mandará al perfil que te corresponde.
    • Mensajes Para poder enviar mensajes a otros usuarios. También se enviarán mensajes de forma automática cuando un usuario pida prestado un producto.
      Admin 'ADMIN'
    • Home En esta página el adminstrador podrá crear a los diferentes usuarios y borrar los que el quiera. En un trabajo futuro se podría imitar este formulario de crear usuarios en una nueva página y hacer que el administrador pueda modificar los roles.
    • Lista de productos Es una tabla con todos los productos del sistema para que cualquiera pueda ver dichos productos de una manera más rápida. No tiene un estilo excesivamente agradable pero cumple su cometido.
      • Los administradores también pueden enviar mensajes.
      Moder 'MODER'
    • Home Pantalla donde aparecen todas las colecciones y productos nuevos para que el moderador les eche un vistado y las valide (aceptar) o las borre (borrar). En caso de borrar un producto se envia un mensaje al propietario explicandole por qué.
      • Admin Moder
      Son perfiles que unen los permisos de los administradores y el de los moderadores.
